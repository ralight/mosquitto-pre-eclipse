Features for initial release (0.1)
==================================

* Must understand and respond to the whole of the mqtt protocol version 3,
  even if it doesn't actually deal with it properly for e.g. wills or retained
  messages.
* Be usable/useful as a broker - subscribe and publish.
* Export some information about the broker through $SYS/ - version, uptime,
  ...?

Things not expected:

* Stable API!
* Bridge support.
* Network interface selection.
* Full $SYS hierarchy.
* Man pages.
